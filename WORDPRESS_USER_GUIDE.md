# WordPress User Guide: How to Add Discovery Tools

This guide shows you how to add the document processing tools (Unlock, Organize, Bates Stamp) to your WordPress website.

## How to Create a Tool Page

You need to create a separate **Page** for each tool you want to display.

### Step 1: Create a New Page
1.  Log in to your WordPress Admin Dashboard.
2.  In the left menu, click **Pages** -> **Add New Application**. (Or just **Add New**).
3.  Give your page a title (e.g., **"Unlock PDF Tool"**).

### Step 2: Insert the Shortcode
In the main content area of the page, type exactly ONE of the following codes.

*   To check if the **Shortcode** block is available, click the **+** (Plus) button and search for "Shortcode". If not, just typing it in a Paragraph block usually works too.

#### Available Shortcodes:

**1. For the Unlock Tool:**
```
[rlg_unlock]
```

**2. For the Organize Tool:**
```
[rlg_organize]
```

**3. For the Bates Stamping Tool:**
```
[rlg_bates]
```

### Step 3: Publish
1.  Click the **Publish** button at the top right.
2.  Click **View Page** to see your new tool in action.

---

## Example: Creating the Bates Stamping Page
1.  Go to **Pages** > **Add New**.
2.  Title: **Bates Stamper**.
3.  Content: Type `[rlg_bates]`.
4.  Click **Publish**.

That's it! Your tool is now ready to use on your website.
